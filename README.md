# Small Office Network Lab

## Objective

Configure a network where PC-A can ping the Server.

## Devices Used

* Router (2911)
* Switch (2960)
* PC-A, PC-B
* Server

## Result

PC-A successfully pinged the Server (10.0.0.2)

## Notes

* First ping may fail due to ARP
* Default gateway must be correct
